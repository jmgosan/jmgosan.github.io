Bucles

Una de las ventajas de los ordenadores es que pueden repetir instrucciones a gran velocidad. Esta repetición es muy interesante para realizar programas en robótica en los que tienes que medir ciertas variables continuamente El bucle for es el más usado.

Mira en la referencia de Processing: https://processing.org/reference/for.html especialmente en la descripción, e intenta entender qué hace el siguiente código:

for (int i = 0; i < 80; i = i+5) 
{
line(30, i, 80, i);
}

Para escribir menos letras

¿Y si lo escribo así?

for (int i = 0; i < 80; i+=5) 
{
line(30, i, 80, i);
}
